# Home Lab

## Overview

This repository documents my enterprise-inspired Linux home lab built to practice System Administration, Networking, Linux, Infrastructure, and troubleshooting.

The goal is to simulate the responsibilities of a Junior System Administrator by deploying, configuring, maintaining, and documenting common enterprise services.

---

## Objectives

- Learn Linux administration
- Practice networking fundamentals
- Deploy production-like services
- Strengthen troubleshooting skills
- Build professional documentation
- Simulate enterprise infrastructure
