# Linux Server Deployment

## Overview
This project demonstrates the deployment and initial administration of an Ubuntu Server within an enterprise-inspired homelab environment. The objective was to build a reliable Linux server that serves as the foundation for infrastructure monitoring, networking, remote administration, and future enterprise services.

## Objectives
- Deploy Ubuntu Server
- Configure basic system settings
- Establish secure remote administration
- Install essential services
- Practice Linux system administration
- Produce professional technical documentation

## Skills Demonstrated
- Ubuntu Server Installation
- Linux Administration
- SSH Remote Administration
- Apache HTTP Server
- User and Group Management
- Linux File Permissions
- Package Management
- Service Management
- Documentation

## Technologies
- Ubuntu Server LTS
- Raspberry Pi 4
- Oracle VirtualBox
- Apache2
- OpenSSH

Successfully deployed a functional Linux server capable of supporting infrastructure monitoring, networking, and future enterprise services within the homelab.
